# Data Analytics Learning Journey

Welcome to my repository tracking my career transition from a biotechnology background into data analytics. 

The purpose of this repository is to act as a structured learning log, project portfolio, and documentation space as I build foundational, industry-standard skills in data science and analytics.

## Learning Roadmap

* **Mathematics & Statistics:** Core concepts for data modeling, probability, and statistical analysis.
* **Excel & Power BI:** Data transformation, advanced formulas, dashboard design, and business intelligence workflows (aligned with PL-300 objectives).
* **SQL:** Relational database management, querying, data manipulation, and ETL processes.
* **Python:** Programming fundamentals for data wrangling, cleaning, and exploratory data analysis (EDA).

## Repository Structure

As I progress, I will organize my scripts, queries, and documentation into the following dedicated folders:

* `01_Mathematics_Stats/`
* `02_SQL/`
* `03_Power_BI_Excel/`
* `04_Python/`

Every tool on this roadmap will be supported by practical mini-projects to document my technical growth step by step.
